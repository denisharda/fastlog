A: Single Hero Button

One big "+250ml" button front and center. Tap to log. Long-press opens custom amount. Maximum simplicity — the whole screen is about one action.

Pros

  • Fastest possible interaction
  • Zero decision fatigue
  • Clean, focused screen

Cons

  • Long-press is hidden/discoverable
  • Locked to one amount size

B: Quick-Tap Row

2-3 preset buttons directly on the main screen (250ml, 500ml, custom). No sheet needed. Primary amount is highlighted, others are secondary.

Pros

  • One tap for common amounts
  • Custom is visible, not hidden
  • Flexible without complexity

Cons

  • Slightly more visual clutter
  • User must choose each time

C: Tap-the-Circle

The center circle IS the button. Tap it to add 250ml. The whole wave area is the touch target. Long-press for custom. The most gesture-driven, minimal UI.

Pros

  • Zero extra UI elements
  • Feels like "filling the glass"
  • Most unique/delightful

Cons

  • Not obvious it's tappable
  • Needs onboarding hint
