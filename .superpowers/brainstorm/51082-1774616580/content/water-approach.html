A: Water-Fill Circle

The circle itself fills with a wave animation from bottom to top. Large intake number in center. Two pill buttons below for quick logging. Tap a button, wave rises, number counts up. Simple, focused, satisfying.

Pros

  • The fill IS the progress — no separate indicator needed
  • Visceral "filling up" feeling
  • Clean 2-button approach, no "Custom" clutter

Cons

  • Less room for extra info inside the circle

B: Big Number + Progress Bar

Hero intake number dominates the screen. Thin progress bar below. Round tap targets (like Apple Watch). Number animates on each tap. Clean, bold, modern. Long-press any button for custom amount.

Pros

  • Feels premium and bold — big typography focus
  • Round buttons are fun to tap (like watch complications)
  • Long-press replaces ugly custom input

Cons

  • Progress bar is less visually exciting than a wave/fill

C: Ring + Single Button

SVG progress ring (like fasting timer). One "Add Water" button that opens a quick radial menu with amounts. Consistent with the timer tab's design language. Tap the ring itself to add the default amount.

Pros

  • Consistent design language with timer tab
  • Single action button is ultra-clean
  • Ring tap = quick add (power user shortcut)

Cons

  • 2 taps to add a specific amount (button → menu)
  • Ring looks similar to timer — less differentiation
